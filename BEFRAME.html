📋 Como funciona o Beframe
  1. Selecione o portal — escolha qual portal da matéria para usar as molduras corretas
  2. Envie o PDF da matéria — as imagens são extraídas e agrupadas automaticamente
  3. Ou monte manualmente clicando no botão + para casos em que o PDF não funcionar
  4. Adicione a capa do artigo usando a seção de capa ou a galeria de imagens
  5. Confira os resultados e use Ajuste fino para mover, cortar, redimensionar ou trocar imagens
  6. Baixe tudo de uma vez como ZIP com o nome do artigo
